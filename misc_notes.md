Windows

Default Split Alt + Shift + D

Split Vertically (Side-by-side): Press Alt + Shift + Plus (+)

Split Horizontally (Top-and-bottom): Press Alt + Shift + Minus (-)

Close the active pane: Ctrl + Shift + W

Linux

Split Vertically (Side-by-Side): Press Ctrl + B, let go, then press % (Shift + 5).

Split Horizontally (Top and Bottom): Press Ctrl + B, let go, then press " (Shift + ' ).


Switch Between Panes: Press Ctrl + B, let go, then press any Arrow Key to jump to the pane in that direction.

Close a Pane: Type exit in the pane you want to close, or press Ctrl + D.

Zoom a Pane: Want to make your Docker logs full screen temporarily? Press Ctrl + B, let go, then press z. Press the exact same sequence to un-zoom.

Create a New Tab: Press Ctrl + B, let go, then press c (for create).

Go to the Next Tab: Press Ctrl + B, let go, then press n (for next).

Go to the Previous Tab: Press Ctrl + B, let go, then press p (for previous).
